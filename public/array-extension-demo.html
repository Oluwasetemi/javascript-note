Safe Array Extension Demo

Extended Array with Sum Method

class ExtendedArray extends Array { sum() { return this.reduce((acc, curr) => acc + curr, 0); } average() { return this.sum() / this.length; } max() { return Math.max(...this); } min() { return Math.min(...this); } } const myArray = new ExtendedArray(1, 2, 3, 4, 5);

Why This is Safe

✅ We extend Array safely by creating a new class

✅ We don't modify the native Array prototype

✅ Other code using Array won't be affected

✅ We get all Array methods plus our custom ones

Dangerous Alternative (DON'T DO THIS):

// ❌ This is dangerous - modifies native prototype Array.prototype.sum = function() { return this.reduce((acc, curr) => acc + curr, 0); };

❌ This affects ALL arrays in the application

❌ Can cause conflicts with other libraries

❌ Can break existing code
